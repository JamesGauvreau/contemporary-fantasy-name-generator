How to Use

If a trait is Forbidden, then applicable names will be rejected. If Favored, applicable names will have greater odds (but not a guarantee) of being chosen. In most cases, it is recommended that, if you pick a non-English language, you still permit English, because these names still occur in the source data. Only exclude English if (1) you are looking for something very strongly of the selected nation(s) or (2) you feel like doing so. Look, I'm not a cop.

Permitting or preferring "Gaelic" does nothing if Irish and/or Scottish names are not checked above.
